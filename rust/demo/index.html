jaxstanv5 — zero-dependency NUTS in your browser

Pick a model from the golden corpus. Each chain runs multinomial NUTS with Stan-style adaptation in its own web worker, inside a wasm build of the Rust core — no server, no bundler, no third-party code.

Serve the repository root (e.g. python3 -m http.server) and run just wasm-release demo-assets in rust/ first so jstan_core.wasm sits next to this page.
